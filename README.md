# Earthhack Template Plugin (1.13+)

This is the template plugin for newgen Earthhack

## Examples:
- Mixin to the Earthhack entrypoint class
- Mixin to the MixinMinecraftClient class
- A simple module with a listener and a setting
- A complete Hud element with some settings and the 2D rendering or a rectangle and an item 
- A simple command

## How to use:
- Clone this repository
- Load the gradle project
- Run `gradle build` to build the plugin
- Copy the jar file from `build/libs` to the `plugins` folder of Earthhack
- Start Earthhack
